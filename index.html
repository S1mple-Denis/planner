Настройка синхронизации

Подключи Firebase — данные будут доступны на всехстройствах

1
Создай проект Firebase
console.firebase.google.com → «Добавить проект» → имя (например mind-planner) → создать.
2
Включи Realtime Database
Build → Realtime Database → «Создать базу данных» → регион Europe → режим «Тест».
3
Скопируй конфиг
Шестерёнка → Project settings → ⟨/⟩ Web → зарегистрируй → скопируй firebaseConfig.
4
Вставь данные
Для постоянной работы вставь данные Firebase прямо в HTML-файл в раздел firebaseConfig — тогда файл работает на всех устройствахбез настройки.
